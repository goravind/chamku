# Backup folder

**To backup with your people and assignees:**
1. In the app: **Settings → Backup & CSV → Export JSON**
2. Save the file as `relocation-tasks.json` in this folder
3. Run `npm run backup` to generate CSV with your tasks and assignees
4. Commit and push

**Sync to GitHub:** Use **Settings → Backup & CSV → Sync to GitHub** to upload directly from the app.

The file `relocation-tasks.csv` is generated by `npm run backup` (uses `relocation-tasks.json` if present, otherwise default checklist).
